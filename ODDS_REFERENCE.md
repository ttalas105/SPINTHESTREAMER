# 1-in-X effective odds for every streamer x element (as displayed in-game)

Effective odds = `streamer.odds * effect.rarityMult`. No effect = base `streamer.odds`.  
Effect rarityMults: Acid x3, Snow x5, Lava x10, Lightning x20, Shadow x40, Glitchy x100, Lunar x200, Solar x500, Void x800.  
Effect roll chances: Acid 5%, Snow 2.4%, Lava 1%, Lightning 0.4%, Shadow 0.15%, Glitchy 0.05%, Lunar 0.02%, Solar 0.007%, Void 0.00375%.
