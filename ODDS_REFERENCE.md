# 1-in-X odds for every streamer × element (luck = 1, no pity)

Rarity roll uses weights: Common 1000, Rare 100, Epic 10, Legendary 1, Mythic 0.1 (scaled by luck^exp).  
Within each rarity, streamer is weighted by 1/odds.  
Effect is rolled in list order: chance = rollChance/rarityMult (Acid 7.5%, Snow 3.7%, Lava 1.78%, Lightning 0.87%, Shadow 0.43%, Glitchy 0.24%, Lunar 0.14%, Solar 0.085%, Void 0.051%).
